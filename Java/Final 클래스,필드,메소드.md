## final

> 최종 선언, 절대 수정 안됨
>
> 클래스 / 필드 / 메소드



### final 클래스

- 부모로 사용 불가한 클래스 **(상속 안 됨)**
- `public final class Info{}`



### final  필드

- 수정 불가 필드 (상수)
- 인스턴스 레벨에서의 상수  `final int PI = 3.14;`
- 클래스 레벨에서의 상수  `static final int MAX_NUM = 100;`



### final 메소드

- **자식이 재정의 할 수 없는** 메소드
- `public final void stop(){ }`
